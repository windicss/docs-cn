# 安装 {#installation}

Windi CSS 为主流工具提供了**最佳的适配**，你可以选择你喜欢的工具并开始使用 Windi CSS。

## 构建工具 {#build-tools}

对于构建工具的支持基本与框架无关，它们适用于大多数框架，无需进行特殊配置。
